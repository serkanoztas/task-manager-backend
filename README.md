# Task Manager Backend

Bu proje, **Node.js**, **Express** ve **MongoDB** kullanılarak geliştirilmiş bir görev yönetim (Task Manager) backend uygulamasıdır. Kullanıcılar görev ekleyebilir, güncelleyebilir, silebilir ve görev durumlarını takip edebilir.

## Özellikler

- Kullanıcı kimlik doğrulama ve yetkilendirme
- Görev ekleme, silme, güncelleme
- Görevleri tamamlandı / tamamlanmadı olarak işaretleme
- Kullanıcı bazlı görev yönetimi
- Validation ile yalnızca izin verilen alanların güncellenmesi

## Teknolojiler

- [Node.js](https://nodejs.org/)
- [Express.js](https://expressjs.com/)
- [MongoDB](https://www.mongodb.com/)
- [Mongoose](https://mongoosejs.com/)
- JWT ile kimlik doğrulama
